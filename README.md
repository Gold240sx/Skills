# Skills

Repo Containing Agent Skills for various tools and Tech Stacks used in my projects.

## App Store Deployment

- [App Store Deployment](appstore-deployment/SKILL.md)

This skill is to help bypass common App Store rejections. It is a work in progress but based off Apple's Developer Documentation, Apple's App Review Guidelines, and Developer Aggreements, as well as dozens, of articles, documentation, app store rejection history and more from the web such as

- Reddit
- Apple's App Store Transparency Dataset
- Stack Overflow
- Quora
- Medium
- onemobile.ai
- decode.agency
- revenuecat.com
  
(This skill was built with Claude 4.6 max, Composer 2, and GPT-5.4 Max)

## WorkOS Pipes Skill

- [WorkOS Pipes](workos-pipes/SKILL.md)

- [WorkOSAuthKitSwift](https://github.com/Gold240sx/WorkOS-Swift)

This skill is to help wire up WorkOS recent Pipes API into any Swift app that uses WorkOSAuthKitSwift (WorkOS Authentication into Swift mobile / Desktop app SDK). It is based off of the working Github Pipe that I've built into a personal MacOS app I've been working on called Devspace. Everything is working seamlessly so far and I plan on integrating many more WorkOS-supported pipes in the future.
